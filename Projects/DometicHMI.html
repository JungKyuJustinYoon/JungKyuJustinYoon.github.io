PLC Hardware Integration & HMI Improvement

Automation Engineering

Overview

Due to excessive wear on the HMI touchscreens, the operators were having trouble using the on-screen acknowledgement buttons. I integrated and programmed physical push buttons, switches, and dials onto the control interface to improve the longevity of the bench and improved user interface. This project taught me how to program using ladder logic and wire hardware into PLC units. By modifying the assembly bench, I was able to make the the hardware integration in locations where it was most convienient for the operator, improving the ergonomics and reducing strain. These modifications led to increased work efficiency and productvity while reducing work-induced strain and risk of injury.

Key Deliverables

  • Siemen's Tiaportal
  • Ladder Logic
  • Hardware integration
